# Hachimi World 基米天堂

Hachimi World is a community-driven open-source meme culture music community

基米天堂是一个由社区驱动的开源梗文化音乐社区

## 关于 API 的使用

基米天堂的开源协议仅适用于项目的源代码。

但本项目的官方服务器 hachimi.world 与其数据不属于开源协议的授权范围，不意味着你可以自由使用。

API 滥用是禁止的。

基米天堂中的二创作品与少量原创作品均由原作者发布。基米天堂只有这些内容的播放权，不拥有代理授权、版权转让等权利。
因此，基米天堂内所有的作品都各自继承其作者的创作声明。若创作者未明确声明，则默认应用各地相关的著作权保护法。这意味着您正常欣赏作品以外的任何其他用途都默认视为未经授权。

基米天堂对你使用作品造成的任何影响不负任何责任。相反，如果您造成了负面影响，你需要对原作者负责。

如果您需要使用基米天堂的内容用于良性用途，请考虑通过 Contribution 方式来完成。
